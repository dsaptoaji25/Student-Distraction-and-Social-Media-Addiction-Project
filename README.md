# Student-Distraction-and-Social-Media-Addiction-Project
Python data analysis that explores the correlation between social media addiction and student distraction. It includes EDA, visualization, and in-depth statistical insights regarding the impact of digital habits on academic performance and psychological health.

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Visualization of usage duration, academic performance, and conflicts scale
4. Insights and recommendations

## Tools & Libraries
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Colab Notebook
- GitHub

## Notes
The Fitbit dataset is available on [Kaggle] (https://www.kaggle.com/datasets/rehan497/students-social-media-addiction)
